# Grade Aggregator

A Java application that calculates weighted academic grades with comprehensive reporting. Reads grading configurations, student scores, computes weighted averages, assigns letter grades with optional +/- adjustment, and generates detailed grade reports.

## Features  
- Calculates weighted averages  
- Comprehensive reporting  
- Supports grading configurations  
- Optional +/- grade adjustments  
- Generates detailed grade reports  

## Getting Started  
To get a local copy up and running, follow these simple steps.
1. Clone the repo
   ```sh
   git clone https://github.com/AOngomefen/GradeAggregator.git
   ```  
2. Open the project in your IDE.

## Configuration  
To configure the grading system, modify the `gradeconfig.txt` file in the `config` directory.

## Grading Scale  
The default grading scale is:
- A: 90-100%
- B: 80-89%
- C: 70-79%
- D: 60-69%
- F: Below 60%
